# Portfolio Profissional
Este repositório contém o código-fonte de um website pessoal desenvolvido para demonstrar competências técnicas em **HTML** e **CSS**, funcionando simultaneamente como um portfólio e currículo digital.

## Visão Geral
O projeto foi concebido para aplicar padrões modernos de desenvolvimento web, priorizando a semântica, o desempenho e a estética visual. Serve como uma vitrina prática de habilidades em design de interfaces e estruturação de documentos.

## Tecnologias e Metodologias

* **HTML5:** Utilização de tags semânticas para acessibilidade.
* **CSS3:** Implementação de layouts modernos, tipografia e estilização avançada
* **Design Responsivo:** Adaptabilidade para diferentes dispositivos e tamanhos de ecrã
